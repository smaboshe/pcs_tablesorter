# pcs_tablesorter

This gem packages the excellent tablesorter.js library for in the Rails 3 asset pipeline.

## Installation

Add this line to your application's Gemfile:

    gem 'pcs_tablesorter'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pcs_tablesorter
